**VR-based teleoperation systems for humanoid robots**

![Teleoperation](../images/teleoperation.png)

## Overview

Development of advanced teleoperation interfaces for robotic avatar systems, enabling intuitive control of humanoid robots through VR devices and specialized peripherals.

## Key Features

- **VR-based interfaces** for immersive robot control
- **Real-time bilateral control** with force feedback
- **Locomotion teleoperation** for humanoid walking
- **Manipulation teleoperation** for dexterous tasks
- **Validated on simulation and real hardware**

## Technologies

- Virtual Reality (VR) devices
- C/C++ real-time control
- YARP middleware
- Quadratic Programming (QP) for motion retargeting
- Walking controllers

## Collaboration

Joint research initiative with **Honda R&D** on their avatar robot platform.

- **Honda avatar robot** - Advanced bilateral teleoperation system
